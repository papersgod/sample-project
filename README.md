# OPIc Sample Project

## Description

This is a sample project demonstrating the OPIc (Oral Proficiency Interview - computer) demo interface. OPIc is an online oral proficiency assessment tool developed by ACTFL (American Council on the Teaching of Foreign Languages) for evaluating speaking skills in various languages.

The project includes a web-based interface for the OPIc testing system, featuring user authentication, multimedia components for video and audio recording, self-assessment tools, and survey functionality.

## Features

- User login interface
- Video and audio recording capabilities
- Self-assessment module
- Survey components
- System check functionality
- Bootstrap-based responsive design
- Analytics integration

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES5/ES6)
- Bootstrap
- jQuery
- Video.js for video playback
- Font Awesome for icons
- Various utility scripts for session management, timeout handling, and RTC detection

## Setup and Installation

1. Clone or download the project files to your local machine.
2. Ensure all files are in the correct directory structure as shown in the workspace.
3. Open `index.html` in a modern web browser (Chrome, Firefox, Safari, or Edge recommended).
4. The application should load and display the login interface.

## Usage

1. The application starts with a login screen where users can enter their credentials.
2. After authentication, users proceed through the oral proficiency interview process.
3. The system includes video recording, audio capture, and interactive survey components.
4. Self-assessment tools allow users to evaluate their performance.

## Notes

- This is a demo/sample implementation and may require backend integration for full functionality.
- Some features like analytics and server-side processing may need additional configuration.
- Ensure microphone and camera permissions are granted for full testing functionality.
- The application uses WebRTC for real-time communication capabilities.
